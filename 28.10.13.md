#Zadanie 1
Zadeklaruje tablice i ja wyswietli

```c
#include <stdio.h>
int main () {
 int a,tab[]={1,3,7,8,9};
 for (a=0; a<=4; a++)
    printf("%i ",tab[a]);
return 0;
} 
```

#Zadanie 2
Zadeklaruje tablice i wyswietli ja w odwrotnej kolejnosci

```c
#include <stdio.h>
int main () {
 int a,tab[]={1,3,7,8,9};
 for (a=4; a>=0; a--)
    printf("%i ",tab[a]);
return 0;
} 
```
