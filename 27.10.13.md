#Zadanie 1 - 
Sprawdza, czy wprowadzona liczba jest wieksza od 5

```c
#include <stdio.h>
int main () {
int n;
printf("Podaj liczbe\n");
scanf("%i",&n);
if (n>5)
printf("Wieksza od 5\n");
else
printf("Mniejsza od 5\n");
return 0;
}
```
